¿Qué es el testing de software?

“Es el proceso que consiste en todas las actividades del ciclo de vida, estáticas y dinámicas, relacionadas con la planificación, preparación y evaluación de productos de software y productos relacionados con el trabajo para determinar que cumplen los requisitos especificados, para demostrar que son aptos para el propósito y para detectar defectos.” ISTQB
